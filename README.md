# Python_101

Collection of basic Python learning scripts and exercises covering a variety of fundamental programming concepts.

## Contents

- Casino simulations (Monte Carlo, roulette)
- List manipulation and for-loop exercises
- Email validation
- Matrix operations
- Sorting algorithms
- Prime number generation
- Statistical averages
- Tkinter GUI basics
- Chilean RUT validation
- JSON and DataFrame operations
- COVID-19 data exploration

## Usage

```bash
python <script_name>.py
```
